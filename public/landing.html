Technology

Built on open science, tuned for India

Sanjeevani's backend is a Python / FastAPI service powered by RDKit for cheminformatics, OPSIN and DECIMER for IUPAC-to-SMILES conversion, and AiZynthFinder for retrosynthetic route planning.

Safety profiling uses SwissADME and ADMETlab for ADMET screening. Patent searches query WIPO, EPO, and Indian patent office APIs. The orchestration layer is a Spring Boot gateway that the React frontend consumes via REST/JSON.

Backend / API Gateway
  • Spring Boot (Java 21)
  • Python / FastAPI microservice
  • PostgreSQL + Redis
Chemistry Engines
  • RDKit · OPSIN · DECIMER
  • AiZynthFinder (retrosynthesis)
  • SwissADME · ADMETlab
Frontend
  • React 18 + Vite
  • TanStack Query + Axios
  • smiles-drawer (2D rendering)
